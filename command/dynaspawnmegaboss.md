# !dynaspawnmegaboss

**Tags:** `dynamis`, `instance`

Instantly spawns the main boss for the specified dynamis zone

**Usage**: `!dynaspawnmegaboss <zoneName>`

**Parameters**:
- `zoneName`: The name of the dynamis zone
  - Options:
    - `bastok`
    - `windurst`
    - `san_doria`
    - `jeuno`
    - `beaucedine`
    - `xarcabard`
    - `valkurm`
    - `qufim`
    - `buburimu`
    - `tavnazia`
