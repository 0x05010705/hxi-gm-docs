# !setweather

**Tags:** `weather`, `zones`

Sets the current weather for the current zone.

**Usage**: `!setweather <weather>`

**Parameters**:
- `weather`: The type of weather to set for the zone
  - Options:
    - 0: None
    - 1: Sunshine
    - 2: Clouds
    - 3: Fog
    - 4: Hot Spell
    - 5: Heat Wave
    - 6: Rain
    - 7: Squall
    - 8: Dust Storm
    - 9: Sandstorm
    - 10: Wind
    - 11: Gales
    - 12: Snow
    - 13: Blizzard
    - 14: Thunder
    - 15: Thunderstorms
    - 16: Auroras
    - 17: Stellar Glare
    - 18: Gloom
    - 19: Darkness
