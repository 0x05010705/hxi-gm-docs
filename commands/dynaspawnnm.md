# !dynaspawnnm

**Tags:** `dynamis`, `spawn`

Spawns the specified NM by the index value for the NM.

**Usage**: `!dynaspawnnm <zoneName> <index>`

**Parameters**:
- `zoneName`: The name of the dynamis zone
  - Options:
    - `bastok`
    - `windurst`
    - `san_doria`
    - `jeuno`
    - `beaucedine`
    - `xarcabard`
    - `valkurm`
    - `qufim`
    - `buburimu`
    - `tavnazia`
- `index`: The index of the NM to spawn
