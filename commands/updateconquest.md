# !updateconquest

**Tags:** `conquest`

Updates all conquest guards.
Values should be updated in the DB first if needed.

**Usage**: `!updateconquest <type>`

**Parameters**:
- `type`: The type of conquest operation to be executed.
  - Options:
    - 0: Conquest Tally Start
    - 1: Conquest Tally End
    - 2: Conquest Update
