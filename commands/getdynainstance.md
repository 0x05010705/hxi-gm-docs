# !getdynainstance

**Tags:** `dynamis`

Prints the instance ID for the specified dynamis zone.

**Usage**: `!getdynainstance <zoneName>`

**Parameters**:
- `zoneName`: The name of the dynamis zone
  - Options:
    - `bastok`
    - `windurst`
    - `san_doria`
    - `jeuno`
    - `beaucedine`
    - `xarcabard`
    - `valkurm`
    - `qufim`
    - `buburimu`
    - `tavnazia`
