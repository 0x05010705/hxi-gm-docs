# !setallegiance

**Tags:** `pvp`

Sets the players allegiance for Ballista/PvP.

**Usage**: `!setallegiance <allegiance> <player>`

**Parameters**:
- `allegiance`: The allegiance to set the player to.
  - Options:
    - 0: Mob
    - 1: Player
    - 2: San D'Oria
    - 3: Bastok
    - 4: Windurst
    - 5: Wyverns
    - 6: Griffons
- `player`: The name of the player to target
