# !getdynavars

**Tags:** `dynamis`

Prints the values of all variables for the specified dynamis zone.

**Usage**: `!getdynavars <zoneName>`

**Parameters**:
- `zoneName`: The name of the dynamis zone
  - Options:
    - `bastok`
    - `windurst`
    - `san_doria`
    - `jeuno`
    - `beaucedine`
    - `xarcabard`
    - `valkurm`
    - `qufim`
    - `buburimu`
    - `tavnazia`
