# !mount

**Tags:** `player`, `fun`

Places the target player on the specified mount ID

**Usage**: `!mount <mountID> (player)`

**Parameters**:
- `mountID`: The ID of the mount to use
  - See Mounts Section below for IDs
- `player`: (Optional) The name of the player to place on mount
